Data Concierge for SAP HANA

    Please select your Data Connection to SAP HANA.

    To do so make a selection for a Data Connection in the selection box underneath the description.

    • Once you have finished your selection you can go ahead by clicking on the Apply Data Connection button.

    Data Connection:

    Please define the source you like to extract.

    Within this dialog you are able to define the source you like to extract. It is possible to extract tables or analytic views from SAP HANA. In order to get a working application please make at least a selection for all fields. The explanation will guide you through the selection steps.

    • green values: These are the selected values
    • white values: These are values which match to your selection
    • grey values: There values doesn't match to your selection

    Explanation

    Fields
    1.) Select a "Type" of source.
    2.) Select a "Schema" for the source.
    3.) Select one "Object" you like to extract.
    4.) Select one or more "Column(s)".
    4.) Select "Dimensions" and "Measures" to define the view.
    5.) Direct Query?

    Overview: Selected Source(s)

    Please have a closer look on the generated Loadscript. In the table below you can check your selection. If you like to watch the autogenerated LoadScript click on the button to expand the script section. If you like to add more data to your self-service scenario you can click on the Add additional Tables button to do so. Once your selection has been finished you can go on by clicking on the Apply Selected Tables button.
